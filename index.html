What is Encoded URL?

  • An encoded URL is a URL where certain characters have been replaced with special codes called percent-encoded characters.
  • These codes are used to represent characters that are not allowed or have special meanings in URLs.
  • Encoded URLs are typically used when passing data via query parameters in a URL or when dealing with special characters.
Example:
  • Original URL: https://example.com/search?q=my search query
  • Encoded URL (using this tool): https%3A%2F%2Fexample.com%2Fsearch%3Fq%3Dmy%20search%20query
  • In the encoded URL, spaces are replaced with %20.
  • In the encoded URL, characters such as : and / are replaced with %3A and %2F, respectively.

What is Decoded URL?

  • A decoded URL is a URL where percent-encoded characters are converted back to their original form.
  • Decoding a URL is necessary to interpret the URL correctly and access the intended resource.
Example:
  • Encoded URL: https%3A%2F%2Fexample.com%2Fsearch%3Fq%3Dmy%20search%20query
  • Decoded URL (using this tool): https://example.com/search?q=my search query
  • In the decoded URL, percent-encoded characters are converted back to their original form.
